导师尽职调查，不靠“感觉流”。

本仓库提供一个用于评估 PhD 导师与实验室的 advisor-research skill。核心视角很简单：选导师很像选一家创业公司的 CEO。先看失败模式，而不是品牌光环。第一步是识别那些会严重伤害博士阶段的关键问题：资金不稳、毕业标准不清晰、文化有毒、去向不佳、署名规范差，或者实验室看起来很强但并不能把学生投入转化为高质量成果。

## 安装为 CLI Skill

克隆本仓库后，你已经拿到了完整的 skill 源码，因为仓库根目录本身就是一个有效的 skill 目录：
- `SKILL.md` 包含核心 skill 指令（Claude Code 与其他兼容 CLI 使用）
- `agents/openai.yaml` 包含可选的 Codex 界面元数据

如果希望自动发现这个 skill，请执行：

```bash
./scripts/install_skill.sh
```

默认会同时安装到：
- `~/.claude/skills/advisor-research`
- `~/.codex/skills/advisor-research`

常用选项：

```bash
./scripts/install_skill.sh --claude
./scripts/install_skill.sh --codex
./scripts/install_skill.sh --skills-dir ~/.your-cli/skills
./scripts/install_skill.sh --copy
./scripts/install_skill.sh --force
```

安装后，重启对应 CLI（或刷新 skills 列表），再用该 CLI 的技能调用语法调用 `advisor-research`。

## 仓库内容

- `SKILL.md`：导师调研 skill 的完整说明。
- `agents/openai.yaml`：可选的 Codex 界面元数据。
- `scripts/install_skill.sh`：把本仓库安装到本地 CLI skill 目录（Claude Code、Codex、或自定义路径）。

## 这个 Skill 会产出什么

- 基于公开证据、可溯源引用的导师档案（dossier）
- 面向学术、工业或创业目标的匹配度评估
- 校友与合作网络映射
- 用于导师沟通与私下学生访谈的问题清单
- 可执行的联系与调研计划

## 从这里开始：先看关键问题

在被名气打动前，先问这些：

- 这位导师能否稳定资助我直到毕业？
- 学生是否能稳定发文并按时毕业？
- 实验室是否心理安全，还是依靠恐惧和过劳运转？
- 署名与贡献分配是否公平？
- 导师是否真的能帮助学生达到其职业目标？
- 一旦进展不顺，实验室有恢复路径，还是学生只能“消失”？

如果其中两项或以上答案偏弱，这就是核心结论，其他都次要。

## 明确的优劣信号

### 强信号（优点）

- 资金强、项目稳定，并且有真实可行的兜底方案
- 学生在学术、工业或两者方向都有清晰结果
- 研究议题有判断力，且有足够基础设施落地执行
- 学生能清楚说明署名规则、里程碑与会议节奏
- 导师反馈及时，能有效帮助项目脱困
- 学生表达自然、被尊重，整体状态健康

### 强信号（缺点）

- 只有名气，没有学生结果
- 口号很大，但资金细节含糊
- 论文很多，但学生所有权不清晰
- 组内成员不敢直说问题
- 毕业周期持续漂移且没有解释
- 导师口头支持工业或非传统路径，但校友证据相反

## 关键建议

- 先保下限，再争上限：安全且高效的组通常优于“名气大但混乱”的组
- 区分导师声誉与学生结果：这两个变量并不等价
- 把所有含糊回答都当作信号：好实验室通常能把机制讲清楚
- 对在读与已毕业学生问同一问题，比较回答差异
- 如果你对学术与工业尚不确定，优先优化“可选择性”
- 不要忽视你的直觉：沟通后若长期感到困惑、紧张或被回避，通常有原因

## 核心评估维度

### 1. 生存、身体与心理健康

- 津贴、学费减免和福利是否足够覆盖基本生活？
- 资金安全是否真实存在，还是一次 grant 决策就会出问题？
- 工作强度是否可持续，还是把 burnout 当常态？
- 组内是否尊重人，还是靠混乱、霸凌或恐惧管理？
- 预期是否足够清晰，让你知道“什么算进展好”？
- 遇到健康、家庭或签证/移民问题时，是否有弹性安排？

### 2. 学术职业结果

- 学生是否稳定发表？发表在什么层级的 venues？
- 毕业去向如何：博后、教职、研究院所，还是不明朗？
- 导师在该方向的声誉与网络强度如何？
- 资金是否足够稳定，支持学生完成学位？
- 学生是否接受了写作、报告与学术社交的系统训练？
- 指导与独立性之间是否平衡？

### 3. 工业职业结果

- 该组是否能提供可迁移到真实工业岗位的技能？
- 校友是否进入了高质量工业岗位？
- 导师是否真心支持非学术路径？
- 学生做实习或产业合作时是否会被设置障碍？
- 实验室在学术圈外雇主中的口碑如何？

### 4. 幸福感与日常匹配

- 你是否真的对研究主题有长期兴趣？
- 日常工作是有投入感，还是长期重复、低回报？
- 学生卡住时，导师是否可触达并愿意介入？
- 组内同学是否合拍、互相支持？
- 地点与生活方式对你是否可持续？
- 工作是否足够有意义，支撑数年投入？

## 跨维度尽职调查

| 维度 | 需要核验的内容 |
| --- | --- |
| 生态与资源 | 学术网络、合作者、算力、设备、数据可得性，以及实验室是否具备落实研究议程的真实基础设施。 |
| 导师职业阶段 | 青年教师可能更投入但强度更高；资深导师可能资源和名望更强，但直接关注更少。 |
| 研究愿景与执行 | 课题品味、关键开放问题、项目质量、推进速度，以及导师能否把想法变成论文而不只是演讲。 |
| 指导风格 | 微观管理或放权、会议频率、预期清晰度，以及风格是否与你的工作方式匹配。 |
| 署名与贡献 | 一作归属规则、跨学生项目分工，以及 postdoc 是否会挤压学生主导权。 |
| 反馈速度 | 论文草稿与项目决策是否足够快，能否赶上关键 deadline。 |
| 职业发展 | 对学术、工业、创业、实习，以及必要时签证/移民流程的支持程度。 |
| 组内文化 | 内部竞争与协作关系、冲突处理方式，以及学生是否显得健康、坦诚且被支持。 |

## 可以问导师的问题

- 你认为本组最有机会真正赢下的未解问题是什么？为什么？
- 如果我现在加入，起步项目会是什么？所需数据、算力、设备和合作者是否已就位？
- 学生每年的资金来源是什么？哪些情况会中断？grant 结束后的兜底方案是什么？
- 具体毕业里程碑是什么？你组里真实的毕业中位时长是多少？
- 你与学生一对一沟通的频率如何？对草稿反馈的常规周转时间是多少？
- 共享项目中的署名与一作归属如何决定？
- 当学生的项目连续几个月没有进展时，通常如何处理？
- 你对工业实习、非学术路径、创业，以及学生目标变化的真实态度是什么？
- 学生遇到健康、家庭、签证等个人冲击时，如何支持且不让学位进程失控？

## 可以问在读或已毕业学生的问题

- 没有对外展示压力时，组内日常到底是什么状态？
- 周末、假期和病假是否真的被尊重？
- 项目失败、停滞或被抢先后，通常会发生什么？
- 学生困难时，导师是帮助、忽视，还是进一步施压？
- 导师在实际中审稿和决策速度有多快？
- 是否有人退组、转组或被边缘化？原因是什么？
- 学生通常对项目有主导权吗，还是被 postdoc/高年级压住？
- 导师在实习、求职、推荐信和签证问题上到底提供多少实质支持？
- 加入前你最希望自己早知道的一件事是什么？
- 哪个在外部看似小问题，进组后会变成大问题？

## 常见红旗

- 资金问题回答含糊、前后不一致，且不同人说法不同
- 学生无法清楚解释署名规则
- 在读学生表达像“背稿”，谨慎且不敢说真话
- 组里把过劳当作投入证明
- 学生长期滞留却缺少强输出或清晰里程碑
- 对工业实习的限制很多，但理由不透明
- 校友去向难以核验或异常单一
- 没人能说清“学生遇到困难时会怎样”

## 结论

把选导师当作尽职调查来做。与导师交流愿景与资源，但关于执行质量、文化氛围和日常体验，优先相信学生证据。通常最优解是：在不透支人生的前提下，给你足够安全、足够成长、也足够兴奋感的那个导师。
